Compression

Batch compressor notebook

Compress a ZIP full of videos using H.265. Typical 40–70% size reduction while keeping quality high. Downloads a ZIP of all compressed videos.

# Upload ZIP → compress all → download ZIP CODEC = 'libx265' # H.265 CRF = 23 # quality PRESET = 'slow' # encoding speed # Saves ~60% file size typically !ffmpeg -i input.mp4 -c:v libx265 \ -crf 23 -preset slow output.mp4
